#' Acquire IPEDS Data for MP#01
#' 
#' This function will acquire and standardize all data for MP#01
#' from IPEDS (https://nces.ed.gov/ipeds/datacenter/DataFiles.aspx)
#' 
#' We're starting in 2010 as the data seems to be reasonably complete 
#' after that point. 
acquire_ipeds_data <- function(start_year=2010, end_year=2024){
    library(tidyverse)
    library(glue)
    
    data_dir <- file.path("data", "mp01")
    
    if(!dir.exists(data_dir)){
        dir.create(data_dir, showWarnings=FALSE, recursive=TRUE)
    }
    
    YEARS <- seq(start_year, end_year)
    
    EFA_ALL <- map(YEARS, function(yy){
        if(yy <= 2022){
            ef_url <- glue("https://nces.ed.gov/ipeds/datacenter/data/EF{yy}A.zip")
            
        } else {
            ef_url <- glue("https://nces.ed.gov/ipeds/data-generator?year={yy}&tableName=EF{yy}A&HasRV=0&type=csv")
        }
        
        ef_file <- file.path(data_dir, glue("ef{yy}a.csv.zip"))
        
        if(!file.exists(ef_file)){
            message(glue("Downloading Enrollment Data for {yy} from {ef_url}"))
            download.file(ef_url, destfile = ef_file, quiet=TRUE, mode="wb")    
        }
        
        read_csv(ef_file, 
                 show_col_types=FALSE) |>
            mutate(year = yy, 
                   # American Indian or Alaskan Native
                   enrollment_m_aian = EFAIANM, 
                   enrollment_f_aian = EFAIANW, 
                   # Asian
                   enrollment_m_asia = EFASIAM, 
                   enrollment_f_asia = EFASIAW, 
                   # Black or African-American, 
                   enrollment_m_bkaa = EFBKAAM, 
                   enrollment_f_bkaa = EFBKAAW, 
                   # Hispanic 
                   enrollment_m_hisp = EFHISPM, 
                   enrollment_f_hisp = EFHISPW, 
                   # Native Hawaiian or Other Pacific Islander 
                   enrollment_m_nhpi = EFNHPIM, 
                   enrollment_f_nhpi = EFNHPIW, 
                   # White
                   enrollment_m_whit = EFWHITM, 
                   enrollment_f_whit = EFWHITW, 
                   # Two or More Races
                   enrollment_m_2mor = EF2MORM, 
                   enrollment_f_2mor = EF2MORW, 
                   # Unknown / Undisclosed Race
                   enrollment_m_unkn = EFUNKNM, 
                   enrollment_f_unkn = EFUNKNW, 
                   # US Non-Resident
                   enrollment_m_nral = EFNRALM, 
                   enrollment_f_nral = EFNRALW, 
            ) |> filter(
                (EFALEVEL %in% c(2, 12)) | (LINE %in% c(1, 15))
                # Per 2024 Data Dictionary, 
                # - EFALEVEL 2 = undergrad 
                # - EFALELVE 12 = grad
                # - Line 1 = first year first time full-time undergrad
                # - Line 15 = first year first time part-time undergrad
            ) |> mutate(level = case_when(
                       EFALEVEL == 2 ~ "all undergrad", 
                       EFALEVEL == 12 ~ "all graduate",
                       LINE %in% c(1, 15) ~ "first year undergrad"
                   )
            ) |>
            select(institution_id = UNITID, 
                   year, 
                   level,
                   starts_with("enrollment_")) |>
            group_by(institution_id, 
                     year, 
                     level) |>
            summarize(across(starts_with("enrollment_"), sum), 
                      .groups = "drop")
        
    }) |> bind_rows()
    
    DESC_ALL <- map(YEARS, function(yy){
        if(yy <= 2022){
            hd_url <- glue("https://nces.ed.gov/ipeds/datacenter/data/HD{yy}.zip")
            
        } else {
            hd_url <- glue("https://nces.ed.gov/ipeds/data-generator?year={yy}&tableName=HD{yy}&HasRV=0&type=csv")
        }
        
        hd_file <- file.path(data_dir, glue("hd{yy}.csv.zip"))
        
        if(!file.exists(hd_file)){
            message(glue("Downloading Institutional Descriptions for {yy} from {hd_url}"))
            download.file(hd_url, destfile = hd_file, quiet=TRUE, mode="wb")    
        }
        
        suppressWarnings(
            read_csv(hd_file, 
                 show_col_types=FALSE, 
                 locale=locale(encoding=if_else(yy==2024, "utf-8", "windows-1252"))) |>
            mutate(year = yy, 
                   INSTNM) |> 
            select(institution_id = UNITID, 
                   institution_name = INSTNM, 
                   state = STABBR, 
                   year)
        )
        
    }) |> bind_rows()
    
    inner_join(EFA_ALL, 
               DESC_ALL, 
               join_by(institution_id == institution_id, 
                       year == year))
}

IPEDS <- acquire_ipeds_data()

##9

The 5 institutions the fraction of white students decrease the most over the period from 2010 to 2020.

Code 
IPEDS |>
  filter(
    year %in% c(2010, 2020),
    level != "first year undergrad"
  ) |>
  mutate(
    total_students = rowSums(across(starts_with("enrollment_")), na.rm = TRUE),
    white_students = enrollment_m_whit + enrollment_f_whit
  ) |>
  filter(total_students > 1000) |>
  group_by(institution_name, year) |>
  summarise(
    white_frac = sum(white_students, na.rm = TRUE) / sum(total_students, na.rm = TRUE),
    .groups = "drop"
  ) |>
  pivot_wider(
    names_from = year,
    values_from = white_frac
  ) |>
  mutate(
    `2010` = as.numeric(`2010`),
    `2020` = as.numeric(`2020`),
    change = `2020` - `2010`
  ) |>
  arrange(change) |>
  head(5) |>
  gt() |>
  cols_label(
    institution_name = md("**Institution**"),
    `2010` = md("**2010**"),
    `2020` = md("**2020**"),
    change = md("**Change**")
  ) |>
  fmt_percent(
    columns = c(`2010`, `2020`, change),
    decimals = 2
  ) |>
  tab_header(
    title = md("**Largest Declines in White Student Share**"),
    subtitle = "Top 5 Institutions, 2010–2020"
  )

The 3 states the fraction of female undergraduates increase the most over the period from 2010 to 2024.

Code 
IPEDS |>
  filter(year %in% c(2010, 2024),
         level == "all undergrad") |>
  mutate(
    total_students = rowSums(across(starts_with("enrollment_"))),
    female_students = enrollment_f_aian + enrollment_f_asia + enrollment_f_bkaa +
      enrollment_f_hisp + enrollment_f_nhpi + enrollment_f_whit +
      enrollment_f_2mor + enrollment_f_unkn + enrollment_f_nral
  ) |>
  group_by(state, year) |>
  summarise(female_frac = sum(female_students) / sum(total_students),
            .groups = "drop") |>
  pivot_wider(names_from = year, values_from = female_frac) |>
  mutate(change = `2024` - `2010`) |>
  arrange(desc(change)) |>
  slice_head(n = 3) |>
  gt() |>
  cols_label(
    state = md("**State**"),
    `2010` = md("**2010**"),
    `2024` = md("**2024**"),
    change = md("**Change**")
  ) |>
  fmt_percent(c(`2010`, `2024`, change), decimals = 2) |>
  tab_header(
    title = "Largest Increases in Female Undergraduate Share",
    subtitle = "Top 3 states, 2010 to 2024"
  )

Final Insights and Deliverable

Building a Diverse Future at Adelphi University

I am the President of CUNY John Jay College fo Criminal Justice, an Institution that prides itself on the importance on diversity on campus. John Jay serves thousands of undergraduate students in NYC with backgrounds from different parts of the world. John Jay role is to foster a supporting community in students to help them grow so they can step out into the real world with confidence. Diversity is essential to preparing students for a global economy through a diverse population of student we are able to strengthen academic discussion, innovation, and career preparation.

Diversity is not just representation its about allowing students to succeed together due to there different backgrounds.That is why it’s important to increase are diverse enrollment for the future.

At CUNY John Jay College for Criminal Justice undergraduate enrollment has about 13,278 students to 11,942 between 2010 and 2024 which reflect a broader-post of pandemic enrollment. As the student body diversity has has been apparent over the the years. With the demographic being made up of Hispanic students, growing from 39.2% in 2010 to 48.3% by 2024. With a large share of students identifying as two or more has slowly increased. We’ve noticed a large shift of minorities becoming the majority of undergraduate students. A comparison with major California public universities shows that the demographic of California Institutes have a more steep decline in white students. California State University San Marcos fell from 44.8% to 24.9%, compared to John Jay’s 10.5% decline in white students.

The numbers are clear, the student body at John Jay College is changing and we must adapt in order for the College to improve educational access and the future of public higher education.
